# Data Extraction and Analysis using MySQL

## 📌 Overview
This project shows how to **extract data** from a MySQL database and **analyze it** using SQL queries.  
We use **MySQL Workbench** to connect to the database, run queries, and view results.

---

## 🛠 Requirements
- MySQL Server (8.0 or higher recommended)
- MySQL Workbench
- A sample database (e.g., `employees`, `sakila`, or your own)
- Basic SQL knowledge

---

## 🔹 Steps

### 1️⃣ Connect to MySQL
1. Open **MySQL Workbench**.
2. Click the **+** button to add a new connection.
3. Enter:
   - **Hostname:** `localhost` (or your server IP)
   - **Port:** `3306` (default)
   - **Username:** `root` (or your DB username)
   - **Password:** (your password)
4. Click **Test Connection** → if successful, click **OK**.
